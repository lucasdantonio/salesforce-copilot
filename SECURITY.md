# Security Policy

## Supported scope

This repository publishes reusable GitHub Copilot assets for Salesforce DX projects.
Please report vulnerabilities that affect the repository itself, bundled scripts, or documented installation guidance.

## Reporting a vulnerability

- Do not open a public GitHub issue for a suspected security vulnerability.
- Email the maintainer or use GitHub private vulnerability reporting if it is enabled for the repository.
- Include the affected file paths, a short impact summary, reproduction steps, and any safe remediation guidance.

## Response expectations

- Reports will be reviewed as quickly as possible.
- Portable fixes are preferred over repo-specific mitigations.
- If the report only affects an example or template, call that out clearly so triage can prioritize correctly.
